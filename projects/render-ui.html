Project: Render UI

Before reading about this project! This project is a Rendering Tool! Not an Add-On!
You want to make a render of your model and you don't know how to do it?

Well, we have the solution for you! Introducing the Render UI!
Let's get into how this rendering tool works! And what can it do!
Show/Hide GUI button is very much self-explanatory, it allows you to expand the render port on small screen devices!
Upload GLTF button will promt you with an upload window, you can upload your GLTF file there!
The Environment button allows you to change the background of the render port! Pretty handy if you want to make a render with a specific scene!
Scaling factor and export work together! The scaling factor allows you to scale your final rendered image by a factor of the input (Default scaling is set to 1)

While the export button allows you to export your render as a PNG file!
