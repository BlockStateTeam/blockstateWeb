Project: Art Enthusiasm Version 3

People love Art enthusiasm Add-On Version 1 and 2. And it's time to announce that the long-awaited update is finally here - introducing Art Enthusiasm Version 3!

There're 2 major changes in this version: Moving away from Beta APIs (Stable) and introducing 2 new tools!
The bucket tools and the Undo Edit are the new tools that are added in this version. The bucket tools allow you to fill in the area with the same color, and the Undo Edit allows you to undo the last edit you made. Their position is below the other tools section.
